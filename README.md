# IndroEconomy
Economy plugin by OMEN44

This is a economy plugin that uses MariaDB to save data.

## Features:
- daily rewards
- OP shop
- Chest shops


Command Syntax:
- () = optional argument
- <> = mandatory argument

Player Commands:
  - /bal (bank/wallet)
  - /pay <player> <amount>
  - /transfer <bank/wallet> <amount>
  - /opshop
  
Admin Commands:
  - /setmoney <bank/wallet> <player> <amount>
